A running dump of things I look up repeatedly or keep getting wrong. These are prime candidates for [[Spaced Repetition]] cards.

---

**Ebbinghaus dates** — keeps slipping. He published *Über das Gedächtnis* in **1885**, not 1895.

**Cowan's number** — working memory capacity is ~**4** chunks (not 7). Miller's original "7 ± 2" was 1956, Cowan's revision was 2001.

**SM-2 algorithm** — the Anki algorithm. Stands for SuperMemo 2. The key parameter is the "ease factor" which starts at 2.5 and adjusts per card.

**Csikszentmihalyi pronunciation** — chick-SENT-me-high

**REM vs SWS for memory:**
- SWS (slow-wave sleep) → declarative memory (facts, concepts)
- REM → procedural + emotional memory
